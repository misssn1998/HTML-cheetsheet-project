HTML Cheetsheet

Basic HTML tags

Tag Name Meaning
<p> Paragraph Defines a Paragraph
<div> Division Divides off a section of code
<a> Anchor Used to link to other locations
<ul> Unordered list Lists elements in bullet points
<li> List Used to create a list
<em> Emphasize Turn text into italic form
<strong> Strong Makes text bold
<br> Break Give breaks to line
